# SportIQ Betting App

Full-stack Node app that:
- Serves a Sofascore-style frontend (public/index.html)
- Provides /api/matches endpoint powered by Betting-API.com
- Uses your SPORTS_API_KEY in the Authorization header

## Running locally

```bash
npm install
SPORTS_API_KEY=your_secret_key_here npm start
```

Then open http://localhost:4000 in your browser.
